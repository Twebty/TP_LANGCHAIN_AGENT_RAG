# TP LangChain Agent + RAG

## 📌 Description
Ce projet illustre l’utilisation de **LangChain** pour construire :
- Un agent simple capable d’exécuter des tâches.
- Un agent basé sur le **RAG (Retrieval-Augmented Generation)** pour enrichir les réponses avec des documents externes.

Il s’agit d’un TP pratique pour comprendre comment intégrer LangChain avec des sources de données et manipuler des agents intelligents.
